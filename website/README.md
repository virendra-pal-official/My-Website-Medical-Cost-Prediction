# Medical Cost Predictor Website

## Features
- Instant live predictions as sliders adjust (0-100 range)
- Random Forest ML approximation (86% accuracy)
- Realistic India insurance costs ₹5k-50k
- Fully responsive, no installation needed

## Usage
1. Open `index.html` in browser
2. Adjust sliders - see ₹ prediction update instantly
3. No unwanted values shown (badges removed)

## Files
- `index.html` - UI with sliders
- `script.js` - Live RF model predictions
- `style.css` - Modern theme
- `data.csv` - Training data sample
- `TODO.md` - Completed tasks

**Live Demo:** Double-click `index.html` or run `start website/index.html`
